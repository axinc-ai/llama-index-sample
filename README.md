# llama-index-sample

## Requirements

- OpenAI API key
- python 3.8 and later
- llama-index (0.6.38.post1)

## Install

```
pip3 install llama-index
```

## Set API key

```
export OPENAI_API_KEY="YOUR_OPENAI_API_KEY"
```

## Create index

```
python3 create_index.py
```

## Query index

```
python3 query_index.py
```
